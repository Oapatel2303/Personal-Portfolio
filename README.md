# 👨‍💻 Ohm's Personal Portfolio

A lightweight, responsive personal portfolio built entirely with Vanilla JavaScript, HTML, and CSS. 

## Features
* **Zero Dependencies:** Built without frameworks to demonstrate core DOM manipulation and modern CSS.
* **Theme Toggling:** Custom CSS variables for seamless light/dark mode switching.
* **Dynamic Rendering:** Content (projects, books, hobbies) is rendered dynamically via JavaScript arrays for easy updating.
* **Custom UI:** Implemented a custom scrollbar and dynamic vertical scroll-progress indicator.
